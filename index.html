Mis Compras es una aplicación móvil diseñada para conectar compradores y vendedores locales en una plataforma moderna, intuitiva y fácil de usar. La aplicación permite explorar productos por categorías, buscar artículos específicos, visualizar detalles completos de cada producto, consultar reseñas de otros usuarios y contactar directamente con los vendedores. Además, cada usuario dispone de un perfil personalizado donde puede gestionar su información, productos publicados y actividad dentro de la plataforma.

Característica principal del proyecto
Catálogo de productos organizado por categorías.
Visualización detallada de productos e imágenes.
Sistema de reseñas y calificaciones.
Perfiles de usuario con gestión de información y productos.
Comunicación directa entre compradores y vendedores.
Interfaz moderna optimizada para dispositivos móviles.

Eccehomo Asadero y Restaurante es una plataforma web diseñada para presentar de manera profesional la oferta gastronómica, turística y de hospedaje del establecimiento. La landing page tiene como objetivo brindar a los visitantes una experiencia visual atractiva, permitiéndoles conocer los servicios, especialidades culinarias, instalaciones y opciones de reserva disponibles desde cualquier dispositivo. El sitio está desarrollado con tecnologías web modernas (HTML, CSS y JavaScript), ofreciendo una interfaz responsiva, rápida y optimizada para mejorar la experiencia del usuario y fortalecer la presencia digital del restaurante.
